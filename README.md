# DATA 612 – Final Project Proposal

## Project Title

**Scalable Movie Recommendation Using Apache Spark: An Item-Based Collaborative Filtering Approach**

## Overview

This proposal outlines my plan to build a scalable movie recommender system using Apache Spark and the MovieLens 1M dataset. The project will implement an Item-Based Collaborative Filtering recommender system and evaluate how Apache Spark performs as the amount of recommendation data increases.

Rather than focusing only on building the recommender system, this project will investigate when the benefits of using Apache Spark outweigh the extra setup and effort required compared to processing the data on a single computer.

## Dataset

- MovieLens 1M Dataset
- Over one million movie ratings
- Approximately 6,000 users
- Approximately 4,000 movies

## Planned Methodology

- Apache Spark (PySpark)
- Item-Based Collaborative Filtering
- Cosine Similarity
- Recommendation Evaluation
- Performance Comparison
- Scalability Analysis

## Project Objectives

- Build a scalable movie recommender system.
- Generate personalized movie recommendations.
- Evaluate recommendation quality.
- Compare how the system performs as the dataset grows.
- Identify when Apache Spark becomes a better solution than using a single computer.
